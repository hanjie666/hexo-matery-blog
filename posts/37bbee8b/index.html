前言

学习到docker，记录下方便以后查看

操作系统

使用VirtualBox虚拟机 + CentOS7系统

安装docker

首先最好更新下软件包跟内核

yum update

安装docker

yum install docker

启动docker

systemctl start docker.service

将docker设置为开机启动

systemctl enable docker

更改docker镜像，加快下载速度

1.创建daemon.json文件

vim /etc/docker/daemon.json

2.复制阿里云镜像粘贴在daemon.json文件

{
 "registry-mirrors":["https://6kx4zyno.mirror.aliyuncs.com"]
}

3.重启docker

systemctl daemon-reload 
systemctl restart docker

docker安装mysql

1.搜索镜像

docker search mysql

2.拉取镜像

docker pull mysql
指定版本
docker pull mysql:5.7

3.查看镜像

docker images

4.运行镜像

docker run --name mysql（自定义容器名） -d(后台运行) images-name(指定镜像模板)

5.查看运行中的容器

docker ps

6.停止容器

docker stop 容器id

7.启动容器

docker start 容器id

8.删除容器

docker rm [IMAGE ID]

9.删除镜像

docker rmi [IMAGE ID]

10.启动一个做了端口映射的tomcat

docker run -d -p 8888:8080 tomcat
-d：后台运行
-p: 将主机的端口映射到容器的一个端口    主机端口:容器内部的端口

11.查看容器的日志

docker logs container-name/container-id

MySql部署

1.拉取MySQL镜像，我这里选择5.7版本

docker pull mysql:5.7

2.创建容器

docker run -p 3306:3306 --name mysql02 -e MYSQL_ROOT_PASSWORD=123456 -d mysql:5.7

-p:端口映射，格式为 宿主机映射端口:容器运行端口

-e:添加环境变量

-d:后台运行

MYSQL_ROOT_PASSWORD:设置root用户的登录密码

3.远程连接宿主机的ip，指定端口号为3306，用户名root，密码123456

tomcat部署

1.拉取镜像

docker pull tomcat

2.创建挂载目录

mkdir -p /mnt/tomcat

3.创建容器

docker run -d --name tomcat_tool -p 8080:8080 -v /mnt/tomcat/webapps:/usr/local/tomcat/webapps --privileged=true a7fa4ac97be4（镜像id）
